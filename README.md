# 🌐 Web AR Virtual Try-On & Learning Platform

## 📌 Overview
A browser-based Augmented Reality platform integrating Virtual Try-On and Interactive Learning. This project is developed in multiple phases to progressively enhance features and scalability.

---

## 🚀 Phase 1 (Completed & Published)
- ✅ Web-based AR using WebXR & Three.js  
- ✅ Virtual Try-On (accessories, furniture)  
- ✅ Interactive 3D learning models  
- ✅ QR-based AR access  
- ✅ Cross-platform compatibility  
- ✅ Research paper published (IJCRT 2025)

## 🔄 Phase 2 (In Progress)
- 🔧 Integration with Supabase / real-time database  
- 🔧 User authentication & profiles  
- 🔧 Save & share AR sessions  
- 🔧 Advanced face & body tracking  
- 🔧 Improved UI/UX  
- 🔧 Product catalog system for e-commerce  
- 🔧 Analytics dashboard  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- AR: WebXR, AR.js  
- 3D: Three.js  
- Backend: Flask / Node.js  
- ML: TensorFlow.js, MediaPipe  
- Database (Phase 2): Supabase / Firebase
